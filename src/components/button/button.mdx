## Variantes

O botão suporta as seguintes variantes:

- **Modernas** (recomendadas): `solid`, `outline`, `text`
- **Especial**: `delete` — atalho semântico para `solid` + cor `negative`
- **Legadas** (retrocompatibilidade): `ghost`, `dashed`, `primary`, `secondary`, `tertiary`, `secondary--white`, `facebook`

Cada variante pode ser combinada com as cores **primary**, **content**, **positive** e **negative**.
