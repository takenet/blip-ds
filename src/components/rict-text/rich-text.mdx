## Customização de estilo

É possível customizar fundo, toolbar e área de edição usando CSS custom properties diretamente no `bds-rich-text`.

<Canvas of={RichTextStories.CustomStyles} />

Partes expostas para estilização:

- `preview`
- `editor`
- `toolbar`
- `toolbar-header`
- `toolbar-buttons`

CSS custom properties disponíveis:

- `--bds-rich-text-background`
- `--bds-rich-text-toolbar-background`
- `--bds-rich-text-toolbar-border-color`
- `--bds-rich-text-toolbar-hover-background`
- `--bds-rich-text-toolbar-button-color`
- `--bds-rich-text-toolbar-button-background`
- `--bds-rich-text-toolbar-button-border-color`
- `--bds-rich-text-editor-background`
- `--bds-rich-text-editor-color`
- `--bds-rich-text-link-color`
