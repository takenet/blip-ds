## Fallback

### Imagem quebrada

Caso a o link da imagem esteja quebrado, uma ilustração para representar esse fallback será renderizado.

### Imagem não encontrada

Caso a propriedade não seja preenchida ou receba vazio, uma ilustração para representar esse fallback será renderizado.
