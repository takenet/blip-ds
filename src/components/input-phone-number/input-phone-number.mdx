## Estrutura

Os inputs são usados para que os usuários insiram e editem números de telefones.

As seguintes propriedades são compartilhadas pelo InputPhoneNumber.

<Controls />
Inputs com código de região
Utilizamos esse estilo de input para facilitar a seleção de número de telefone dentro da plataforma, em áreas que podem receber esse tipo de conteúdo como nossa área de account ou nos templates de mensagem do WhatsApp.

## Especificações do componente:

• Ao selecionar o país o código com o "+" é inserido no input.

• Aceitar somente caracteres de números.

• Possibilitar definir o componente como required.

• Bandeira do Brasil Default.

• Depois de selecionado o país o código não será editável.
