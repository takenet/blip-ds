## Estrutura

O card é um componente usado para fornecer informações sobre atualizações e notificações.

Para a construção do componente Card você terá tags obrigatórias e outras opcionais para compor o layout. São elas:

<ul>
  <li>bds-card-header</li>
  <li>bds-card-body</li>
  <li>bds-card-footer</li>
  <li>bds-card-title</li>
  <li>bds-card-subtitle</li>
</ul>
