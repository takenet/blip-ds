## Instalando o projeto

Recomendamos usar npm para instalar a biblioteca de componentes do blip-ds.

<code>npm install blip-ds</code>

<br></br>
## Recomendações de bibliotecas

Recomendamos a instalação de pacotes que vão incrementar na utilização do Design System.

<code>npm install sass</code>

<br></br>
