# Estilização no projeto

<br></br>
No framework React, recomendamos duas formas de estilizar seus elementos em tela.

## Estilizar via css-in-js
De forma bem similar ao uso do styled component, existe a possibilidade de passar uma prop para o style do componente ou elementos superiores e definir o CSS-in-JS, como uma prática bem comum no framework react.

<img src={image} alt="Imagem de exemplo" />

```jsx
import React from "react";

const teste = {
  width: '500px',
  margin: 'auto',
  marginTop: '36px',
  padding: '32px',
}

export default class Test extends React.Component {
  render() {
    return (
        <bds-paper style={teste}>
          <bds-typo>Teste de uma prop para estilizar esse componente css-in-js</bds-typo>
        </bds-paper>
    );
  }
}
```

## Estilizar via className no scss
A forma mais apropriada para estilizar qualquer elemento em tela seria via className.

<img src={image2} alt="Imagem de exemplo" />

### Arquivo .jsx

```jsx
import React from "react";

export default class Test extends React.Component {
  render() {
    return (
        <bds-paper class="teste">
          <bds-typo>teste de estilização via SCSS</bds-typo>
        </bds-paper>
    );
  }
}
```
### Arquivo .scss

```jsx
.teste {
  width: 500px;
  margin: 36px auto;
  padding: 32px;
}
```
